## ‚ÑπÔ∏è TTL files
This folder contains the **TTL exports** of all Arabic rhetorical devices and taxonomy items included in **Arabic Rhetorical Device Taxonomy (v0.1.1)**.

* Each file (`Qxx.ttl`) corresponds to one Arabic rhetorical device or taxonomy item.  
* Each file (`Qxx.ttl`) contains RDF triples in compact Turtle syntax.
* Each file (`Qxx.ttl`) is exported directly from [**BalaghaBase.org**](https://balaghabase.org) (custom Wikibase instance). 
* TTL is widely used for: semantic web research, ontology engineering, digital humanities pipelines.  
* All files use the **schema.org** namespace and standard RDF prefixes.  
* Validated for FAIR compatibility and consistency across formats.
