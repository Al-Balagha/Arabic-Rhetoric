The systematic, hierarchical classification of **Arabic rhetorical devices** published by the [Encyclopedia of Arabic Rhetoric](https://al-balagha.com).

## ‚ÑπÔ∏è About this folder
This folder contains the taxonomy of Arabic rhetorical devices in self-contained, standalone HTML format.

The folder structure is as follows:

* [Encyclopedia_of_Arabic_Rhetoric.html](Encyclopedia_of_Arabic_Rhetoric.html) - the encyclopedia homepage
* README.md - this file
* [Arabic_Rhetorical_Devices](Arabic_Rhetorical_Devices) - a folder containing the taxonomy in the form of HTML files
* [Supporting_Pages](Supporting_Pages) - a folder containing HTML files giving background information about Arabic rhetoric and the Encyclopedia. It also contains a glossary of commonly used terms.
