# [Arabic Rhetorical Device Taxonomy (v0.1.1)](https://github.com/Al-Balagha/Arabic-Rhetoric/tree/v0.1.1)
### A project of the [Encyclopedia of Arabic Rhetoric](https://al-balagha.com)

The systematic, hierarchical classification of **Arabic rhetorical devices** published by the [Encyclopedia of Arabic Rhetoric](https://al-balagha.com).

This folder contains **machine-readable serialisations** of the **Arabic Rhetorical Device Taxonomy (v0.1.1)**.

Each subfolder corresponds to one structured data format used in digital humanities, linked-data ecosystems, and computational linguistics research.

&nbsp;
### üìÅ Folder structure

| Folder | Format | Purpose |
|--------|--------|---------|
| `/json/` | JSON | Lightweight structured representation, compatible with scripting and data pipelines. |
| `/jsonld/` | JSON-LD | Web-native linked data using `schema.org` and Wikibase context. |
| `/ttl/` | Turtle (TTL) | Human-friendly RDF format widely used in DH and ontology work. |
| `/rdf/` | RDF/XML | Classical W3C RDF serialisation for ontology editors and graph stores. |

Each folder includes an **`index` file** listing all Q-items available in that format.

&nbsp;
### ‚ÑπÔ∏è About this folder
* All files were exported directly from [**BalaghaBase.org**](https://balaghabase.org) (custom Wikibase instance).
* The content of each Q-item is identical across formats; only the serialisation differs.  
* This structure implements FAIR principles by ensuring Findability, Accessibility, Interoperability, and Reusability.
* The JSON-LD, TTL, and RDF/XML formats provide full compatibility with semantic web tools.
